# NeoFrame Home Assistant Add-on

![Supports amd64 Architecture][amd64-shield]
![Supports aarch64 Architecture][aarch64-shield]
![Supports armv7 Architecture][armv7-shield]

Dithers and pushes images to a Good Display NeoFrame 13.3" Spectra 6 e-ink display via ESP32. Polls the InkyPi add-on for image changes and automatically updates the frame.

For full documentation, see [DOCS.md](DOCS.md).
